# Arkanoid - Proyecto M2

Juego arcade desarrollado en Python utilizando la biblioteca Pygame. El objetivo es destruir todos los bloques de la pantalla sin dejar caer la bola.

## Requisitos
* Python 3.10 o superior
* Pygame (`pip install pygame`)

## Instalación y Ejecución
1. Clona este repositorio o descarga el ZIP.
2. Navega a la carpeta del proyecto.
3. Ejecuta el juego indicando el nivel a cargar:
   ```bash
   python arkanoid_game.py niveles/demo.txt
   python arkanoid_game.py niveles/nivel1.txt
   python arkanoid_game.py niveles/nivel2.txt
